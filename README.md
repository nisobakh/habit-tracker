habit-tracker
Lightweight habit and wellness tracker concept.

Overview  
This project is a simple prototype for a daily habit tracking application. It helps users record progress for recurring goals such as exercise, reading, and sleep consistency.

Features  
- Log and store daily habits with timestamps  
- Calculate weekly consistency metrics  
- Display progress summaries for each habit
